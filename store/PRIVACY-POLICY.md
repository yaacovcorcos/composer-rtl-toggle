# Privacy Policy for Composer RTL Toggle

Last updated: 2026-04-11

Composer RTL Toggle does not collect, transmit, sell, or share personal information.

## What the extension does

The extension adds a small RTL/LTR toggle inside the composer on supported websites so the user can switch writing direction in the prompt box.

## Data the extension stores

The extension stores only one type of local setting:

- the selected writing direction (`LTR` or `RTL`) for each supported website

This information is stored locally using Chrome's extension storage.

## What the extension does not do

- It does not collect prompts or message content
- It does not read or store account credentials
- It does not send data to external servers
- It does not use analytics, tracking, or advertising
- It does not sell or share user data

## Permissions

The extension uses the `storage` permission only to remember the user's preferred writing direction on supported websites.
